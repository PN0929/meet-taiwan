# 📋 Templates - 模板檔案

這個資料夾包含各種模板,幫助你更有效率地工作。

## 可用模板

### 🆕 [NEW_CONVERSATION_STARTER.md](./NEW_CONVERSATION_STARTER.md)
**用途:** 開啟新的 Claude 對話時使用

當現有對話接近上限,或需要開新對話時:
1. 打開這個檔案
2. 複製內容
3. 貼到新的 Claude 對話
4. 更新 "目前進度" 和 "當前任務"
5. Claude 就能理解整個專案!

### 📊 [PROGRESS_REPORT_TEMPLATE.md](./PROGRESS_REPORT_TEMPLATE.md)
**用途:** 記錄階段性進度

定期 (例如每週) 填寫進度報告:
- 記錄完成的事項
- 追蹤進行中的任務
- 規劃下一步
- 記錄重要決策

### 🐛 [ISSUE_TEMPLATE.md](./ISSUE_TEMPLATE.md)
**用途:** 回報問題或建議

在 GitHub 開 Issue 時使用,提供完整資訊方便排查。

## 如何使用

1. 點擊模板檔案
2. 點 "Raw" 按鈕查看原始內容
3. 複製內容
4. 依據實際情況填寫
5. 使用!
