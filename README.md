# LOCAL FRIENDS TAIWAN 台灣在地好友

> 獨立資訊平台 - 連結外國旅客與台灣在地文化體驗

## 專案狀態

🚧 **建置中** - MVP 開發階段

## 技術架構
```
Google doc (內容管理,中文) 
  ↓
Claude API (自動翻譯)
  ↓
Next.js (網站)
  ↓
Vercel (部署)
```

## 核心概念

- **定位:** 資訊策展平台,非預訂網站
- **內容:** 深度文化體驗
- **差異:** 連結真實的人,而非標準化行程

## 工作流程

1. google doc, sheet 寫內容(中文)
2. 翻譯成英文
3. 推送更新
4. 網站自動部署
