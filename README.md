# Meet Taiwan 遇見台灣

> Find Your People in Taiwan

## 🌟 專案簡介

Meet Taiwan 是一個獨立的資訊策展平台,專注於連結外國旅客與台灣真實的在地文化體驗。

**這不是預訂平台,而是文化連結的橋樑。**

我們整理並介紹:
- 🏘️ 社區走讀與在地生活
- 🎨 傳統工藝與創作者
- 🌾 農村與永續生活
- 🏔️ 原住民文化體驗
- 🤝 社會議題與 NGO 參訪
